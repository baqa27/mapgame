# Review

## Saran Diterima

1. Reward kurang menarik, sebaiknya reward dibuat lebih menarik.

Alasan diterima: Reward pada setiap mode sebelumnya masih terlalu umum dan kurang memberi rasa pencapaian kepada pemain. Dengan penambahan reward seperti unlock dialog rahasia, item khusus, true ending, dan free mode, pemain menjadi lebih termotivasi untuk menyelesaikan permainan hingga akhir.

3. Setelah hard mode sebaiknya ada mode jalan-jalan santai tanpa story ataupun teka-teki.

Alasan diterima: Saran ini sesuai untuk dijadikan bonus setelah seluruh game selesai. Free mode memberikan kesempatan pemain menikmati suasana desa tanpa tekanan investigasi atau ancaman, sekaligus menambah nilai eksplorasi dan replayability game.

5. Sebaiknya tambahkan suara kentongan saat terjadi sebuah aksi yang menyebabkan dampak, apalagi di hard mode.

Alasan diterima: Suara kentongan sangat sesuai dengan tema ronda malam desa. Penambahan efek suara dapat meningkatkan suasana tegang, memperkuat nuansa horor, dan membantu memberikan tanda ketika terjadi peristiwa penting dalam permainan.

## Saran Ditolak

2. Sebaiknya minimap hilang atau disamarkan saat di mode hard biar lebih menantang.

Alasan ditolak: Hard mode sudah memiliki tingkat kesulitan tinggi melalui investigasi bercabang, tekanan sosial, dan gangguan mistis. Jika minimap dihilangkan, pemain berpotensi lebih sering tersesat dan kesulitan navigasi sehingga fokus terhadap inti gameplay dan alur cerita menjadi berkurang.

4. Sebaiknya ada edukasi aksi, misal salah tuduh bikin uang jimpitan berkurang.

Alasan ditolak: Konsekuensi salah tuduh sudah diterapkan melalui penurunan kepercayaan warga, perubahan jalur cerita, dan kemungkinan gagal ending. Jika ditambah pengurangan uang jimpitan, sistem hukuman akan terasa terlalu berat dan dapat mengurangi kenyamanan bermain.
